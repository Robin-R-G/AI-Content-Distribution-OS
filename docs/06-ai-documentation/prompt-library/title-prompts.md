# Title Prompts Library

## Overview

Title generation prompts for blogs, videos, newsletters, and articles. Optimized for click-through rates, SEO, and platform-specific algorithms.

---

## Blog Post Title Prompt

### System Prompt

```
You are an expert headline writer specializing in blog post titles. You understand SEO, click-through rate optimization, and reader psychology. You write titles that are compelling, accurate, and keyword-rich.

Key rules:
- Front-load primary keywords for SEO
- Use power words that drive clicks
- Keep under 60 characters for SEO display
- Use numbers when appropriate
- Create curiosity without clickbait
- Match title to content (no false promises)
```

### User Prompt Template

```
Generate 5 blog post titles for:

Topic: {topic}
Niche: {niche}
Target Audience: {audience}
Primary Keyword: {primary_keyword}
Secondary Keywords: {secondary_keywords}
Content Type: {content_type}
Tone: {tone}

Blog Title Rules:
- Max 60 characters for SEO
- Front-load primary keyword
- Use power words
- Create curiosity
- Match search intent
```

### Title Formulas

#### How-To Titles
```
Formula: How to [Achieve Result] [Timeframe/Method]

Examples:
- How to Double Your Email List in 30 Days (Without Paid Ads)
- How to Start a Podcast With Zero Experience in 2024
- How to Build a Personal Brand That Actually Converts
```

#### Listicle Titles
```
Formula: [Number] [Adjective] Ways to [Achieve Result]

Examples:
- 7 Proven Ways to Go Viral on LinkedIn (Backed by Data)
- 15 Easy Meal Prep Ideas That Actually Taste Good
- 10 Silent Signs You're Burning Out (And How to Fix It)
```

#### Question Titles
```
Formula: Are You Making These [Topic] Mistakes?

Examples:
- Are You Making These Instagram Engagement Mistakes?
- Is Your Website Costing You Customers? (Here's How to Tell)
- Still Doing SEO the Old Way? Here's What's Changed
```

#### Comparison Titles
```
Formula: [Option A] vs [Option B]: Which Is Better for [Goal]?

Examples:
- ConvertKit vs Mailchimp: Which Email Platform Is Best for Creators?
- React vs Vue: Which Framework Should You Learn in 2024?
- Remote Work vs Office: The Data Doesn't Lie
```

#### Guide Titles
```
Formula: The Complete Guide to [Topic] [Year]

Examples:
- The Complete Guide to Instagram Reels in 2024
- The Ultimate Guide to YouTube SEO
- The Beginner's Guide to Starting an Online Business
```

## YouTube Video Title Prompt

### System Prompt

```
You are a YouTube SEO expert. You write titles that maximize click-through rates while accurately representing video content. You understand YouTube's algorithm prioritizes watch time and engagement.

Key rules:
- Max 60 characters (displayed fully)
- Front-load keywords
- Create curiosity gap
- Use caps strategically (1-2 words max)
- Include year when relevant
- Avoid clickbait (hurts watch time)
- Test multiple title patterns
```

### User Prompt Template

```
Generate 5 YouTube video titles for:

Topic: {topic}
Video Type: {video_type}
Target Audience: {audience}
Primary Keyword: {primary_keyword}
Key Benefit: {key_benefit}
Video Length: {video_length}

YouTube Title Rules:
- Max 60 characters
- Front-load primary keyword
- Create curiosity without clickbait
- Use power words
- Include year if evergreen topic
```

### YouTube Title Patterns

#### Tutorial/Tutorial
```
Pattern: How to [Action] [Context] (Even if [Obstacle])

Examples:
- How to Edit Videos Fast (Even If You're a Complete Beginner)
- How to Grow on YouTube in 2024 (Step-by-Step Tutorial)
- How to Build a Website in 30 Minutes (No Coding Required)
```

#### Listicle
```
Pattern: [Number] [Topic] [Year] | [Benefit]

Examples:
- 10 YouTube Tips Nobody Tells You | Grow Faster in 2024
- 7 Video Editing Hacks That Will Save You Hours
- 5 YouTube Shorts Ideas That Go Viral Every Time
```

#### Review/Comparison
```
Pattern: [Product] Review: [Honest Assessment] ([Year])

Examples:
- iPhone 15 Pro Review: Is It Worth the Upgrade? (2024)
- Notion vs Obsidian: The Ultimate Comparison (2024)
- Best Budget Microphones 2024: Honest Review
```

#### Story/Case Study
```
Pattern: I [Did Action] for [Timeframe] — Here's What Happened

Examples:
- I Posted on YouTube Every Day for 365 Days — Here's What Happened
- I Switched to Notion for 6 Months — My Honest Review
- I Spent $10,000 on Facebook Ads — Here's What I Learned
```

## Newsletter Subject Line Prompt

### System Prompt

```
You are an email marketing expert specializing in newsletter subject lines. You understand open rate optimization, A/B testing, and inbox placement.

Key rules:
- Max 50 characters for mobile display
- Front-load value/curiosity
- Avoid spam triggers (ALL CAPS, excessive punctuation)
- Use personalization when available
- Create urgency without being misleading
- A/B test two variations
```

### User Prompt Template

```
Generate 10 newsletter subject lines for:

Topic: {topic}
Newsletter Type: {newsletter_type}
Audience: {audience}
Goal: {goal}
Key Content: {key_content}

Newsletter Rules:
- Max 50 characters (mobile-optimized)
- Front-load value
- Avoid spam triggers
- Create urgency/curiosity
- Include A/B test pair
```

### Subject Line Patterns

#### Curiosity Gap
```
Pattern: [Surprising Fact] or [Question That Needs Answering]

Examples:
- The marketing mistake costing you $10K/month
- Why your emails aren't getting opened (fix inside)
- What nobody tells you about hiring
- The one metric that actually matters
```

#### Value-Driven
```
Pattern: [Specific Benefit] + [How/When]

Examples:
- Double your productivity with this simple system
- 3 templates that saved me 20 hours last week
- The exact script I use for cold outreach
- How to land your first 10 clients
```

#### Urgency/FOMO
```
Pattern: [Time Sensitivity] + [Opportunity]

Examples:
- Last chance: Free workshop tomorrow
- This disappears Friday (not kidding)
- You're missing this (and it's free)
- Only 24 hours left to...
```

#### Personal/Story
```
Pattern: [Personal Story] + [Lesson]

Examples:
- I got fired. Here's what I learned.
- The email that changed my career
- Why I almost quit this week
- My biggest mistake this year
```

## SEO Title Tag Prompt

### System Prompt

```
You are an SEO specialist. You write title tags that maximize click-through rates in search results while maintaining keyword optimization.

Key rules:
- Max 55-60 characters (Google truncates after)
- Primary keyword in first position
- Include secondary keyword naturally
- Use power words for CTR
- Match search intent
- Brand name at end (if space allows)
```

### User Prompt Template

```
Generate 5 SEO title tags for:

Page: {page_type}
Primary Keyword: {primary_keyword}
Secondary Keywords: {secondary_keywords}
Brand Name: {brand_name}
Search Intent: {search_intent}

SEO Rules:
- Max 60 characters
- Primary keyword first
- Power words for CTR
- Match search intent
- Include year if relevant
```

### Title Tag Patterns

#### Informational Intent
```
Pattern: [Primary Keyword] — [Benefit] | [Brand]

Examples:
- Email Marketing Tips: 15 Ways to Boost Open Rates | Mailchimp
- Beginner's Guide to SEO (2024) — Everything You Need to Know
- What Is Content Marketing? A Complete Guide | HubSpot
```

#### Commercial Intent
```
Pattern: [Product] Review: [Pros] | [Year]

Examples:
- Best Email Marketing Software 2024: Tested & Ranked
- ConvertKit Review: Best for Creators? (Honest Take)
- Top 10 Project Management Tools Compared | 2024
```

#### Transactional Intent
```
Pattern: [Action] + [Benefit] + [CTA]

Examples:
- Buy Email Templates — Save 50% Today
- Start Free Trial: No Credit Card Required
- Download Free SEO Checklist (Instant Access)
```

## Cross-Platform Title Variables

### Input Variables

| Variable | Type | Description |
|----------|------|-------------|
| `{topic}` | string | Main topic |
| `{niche}` | string | Industry/category |
| `{audience}` | string | Target audience |
| `{primary_keyword}` | string | Main SEO keyword |
| `{secondary_keywords}` | string[] | Related keywords |
| `{content_type}` | enum | how-to, listicle, guide, review, story |
| `{tone}` | enum | professional, casual, humorous, educational |
| `{key_benefit}` | string | Primary value proposition |
| `{data_points}` | string[] | Statistics or data to include |
| `{year}` | number | Current year for relevance |

### Character Limits by Platform

| Platform | Display Limit | SEO Limit | Notes |
|----------|--------------|-----------|-------|
| Blog (H1) | 70 chars | 60 chars | H1 tag |
| YouTube | 100 chars | 60 chars | Search results truncation |
| LinkedIn | 150 chars | 70 chars | Mobile display |
| Newsletter | 50 chars | 50 chars | Mobile inbox |
| Twitter | 70 chars | 70 chars | Preview card |
| Pinterest | 100 chars | 100 chars | Pin title |
| Facebook | 255 chars | 100 chars | Post title |

### Power Words by Category

| Category | Words |
|----------|-------|
| Curiosity | secret, hidden, surprising, unexpected, little-known |
| Urgency | now, today, fast, quick, instant, immediate |
| Value | free, proven, tested, ultimate, complete, essential |
| Emotional | amazing, incredible, heartbreaking, shocking, inspiring |
| Authority | expert, professional, scientific, research-backed |
| Numbers | 10, 7, 5, 3, 15, 21, 100, data-backed |
| Action | boost, maximize, transform, master, conquer |
