## Instagram Hashtag Prompt

### System Prompt

```
You are an Instagram hashtag strategist. You understand that hashtags are discovery tools, not decoration. You generate hashtag sets that balance reach, relevance, and competition.

Key rules:
- Use 20-30 hashtags on Instagram (max allowed)
- Mix popularity levels: 3-5 large (1M+), 10-15 medium (100K-1M), 5-10 niche (10K-100K)
- All hashtags must be relevant to the content
- Avoid banned or flagged hashtags
- Rotate hashtag sets to avoid shadowbanning
- Place in first comment or caption end
```

### User Prompt Template

```
Generate Instagram hashtags for:

Topic: {topic}
Niche: {niche}
Target Audience: {audience}
Content Type: {content_type}
Account Size: {account_size}
Goal: {goal}

Hashtag Strategy:
- Total: 25-30 hashtags
- Large (1M+ posts): 3-5
- Medium (100K-1M): 10-15
- Small/Niche (10K-100K): 5-10
- Include 3-5 branded hashtags if applicable
- No banned hashtags
```

## Hashtag Rotation System

### Purpose
Prevent shadowbanning by rotating hashtag sets.

### Rotation Strategy
```typescript
interface HashtagSet {
  id: string;
  hashtags: string[];
  lastUsed: Date;
  usageCount: number;
  avgEngagement: number;
}

const ROTATION_RULES = {
  maxConsecutiveUses: 3,
  minDaysBetweenUses: 2,
  maxSetSize: 30,
  refreshRate: 'weekly',
};
```

### Rotation Template
```
Set A (High Engagement): Top-performing hashtags
Set B (Discovery): Focus on niche/micro hashtags
Set C (Trending): Current trending hashtags
Set D (Experimental): New/test hashtags

Rotate: A → B → C → D → A
```

## Banned/Flagged Hashtags Reference

### Always Avoid
```
#adulting (flagged)
#adult (banned)
#narcissist (flagged)
#alone (flagged)
#bored (flagged)
```

### Check Before Using
Always verify hashtags using:
- Instagram's search (check if posts show)
- Hashtag research tools
- Community databases

### Safe Alternatives
```
Instead of #adulting → #growingup #lifelessons
Instead of #alone → #solitude #timetomyself
Instead of #bored → #boredom #needinspiration
```

## Branded Hashtag Strategy

### Creating Branded Hashtags
```
Formula: [BrandName] + [Action/Community]

Examples:
- Nike: #JustDoIt, #NikeRunning
- Glossier: #GlossierIRL, #IntoTheGloss
- Your Brand: #[BrandName]Community, #[BrandName]Tips
```

### Branded Hashtag Rules
1. Keep it short and memorable
2. Make it unique (search first)
3. Encourage user-generated content
4. Include in bio and posts consistently
5. Create challenge/contest hashtags quarterly
