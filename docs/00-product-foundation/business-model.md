# Business Model

## Model Overview

**SaaS + Marketplace**

A subscription-based software platform with an integrated marketplace for plugins, templates, and AI credits. The core platform generates recurring revenue; the marketplace creates network effects and additional monetization.

## Core Business Logic

### Value Proposition

| Segment | Value | Willingness to Pay |
|---------|-------|---------------------|
| Solo Creator | Time savings, growth tools | $5-$15/month |
| Growing Creator | Systems, team collaboration | $12-$25/month |
| Agency | Client management, efficiency | $36-$100/month |
| Brand Team | Governance, ROI demonstration | $100-$500/month |
| Developer | Extensibility, API access | $20-$50/month |

### Unit Economics

| Metric | Target | Benchmark |
|--------|--------|-----------|
| CAC (Solo Creator) | $25 | Industry avg: $50 |
| CAC (Agency) | $450 | Industry avg: $800 |
| LTV (Solo Creator) | $180 | Industry avg: $300 |
| LTV (Agency) | $18,000 | Industry avg: $25,000 |
| LTV:CAC Ratio | 7:1+ | Industry target: 3:1 |
| Payback Period | <6 months | Industry avg: 12 months |
| Monthly Churn (Solo) | <8% | Industry avg: 10% |
| Monthly Churn (Agency) | <2% | Industry avg: 3% |

### Cost Structure

| Category | % of Revenue | Description |
|----------|-------------|-------------|
| AI API Costs | 15-20% | OpenAI, Gemini, Claude API usage |
| Infrastructure | 10-15% | Cloudflare, Supabase, Redis |
| Payment Processing | 3-5% | Stripe, Razorpay fees |
| Customer Support | 8-10% | Support team and tools |
| Marketing | 20-25% | Acquisition and brand building |
| Development | 25-30% | Engineering team |
| G&A | 10-12% | Operations, legal, admin |
| **Target Margin** | **15-20%** | Net profit margin at scale |

## Growth Strategy

### Phase 1: Acquisition (Months 1-6)
- Free tier drives signups
- Content marketing and SEO
- Creator community partnerships
- Product-led growth (PLG)

### Phase 2: Activation (Months 3-9)
- Onboarding optimization
- AI first-value within 5 minutes
- Template library for quick starts
- Integration wizards

### Phase 3: Revenue (Months 6-12)
- Free-to-paid conversion: 5-8%
- AI credit upsell
- Template marketplace
- Agency onboarding

### Phase 4: Expansion (Months 9-18)
- Cross-sell (solo → agency)
- Upsell (credits, templates)
- Marketplace commissions
- White-label licensing

### Phase 5: Ecosystem (Months 12-24)
- Plugin marketplace revenue
- API usage billing
- Partner integrations
- Developer community
